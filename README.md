# Task-Scheduling-App
This is a simple task scheduling application that allows users to manage their daily tasks by adding, viewing, and organizing them based on date and time. The user can input a task along with its scheduled date and time, and the system stores it for future reference.

The application provides multiple viewing options, including displaying all tasks for a specific date entered by the user, as well as automatically showing tasks for today and upcoming days using the system’s current time. Users can easily navigate through these options and exit the application when finished.

This project demonstrates basic concepts of data handling, date/time management, and user interaction in a console-based environment.
